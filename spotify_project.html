Spotify Data Analysis

After requesting my personal data from Spotify, I collected individual track features using Spotify API's. I cleaned and organized my data before performing some data analysis. It was my first time collecting my own data and also my first time using Spotify's API.
As of 8/22/2021 I have finished the first version. I am currently waiting on more data from Spotify before performing some analysis between my listening habits and a friend.

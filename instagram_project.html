Instagram Clone

This is my second backend project, an instagram clone using Laravel. It is the final project of a course YouTube that I'm using to study PHP. The features are: user creation, authentication system, users can follow each other, upload images, edit their profile info, a feed to display the posts of the users they followed, visiting an individual post. The frontend is not responsive since this is a backend project. Bootstrap is used to create the front end, vue js for the follow button component and SQLite to store user data. While this project is mostly completed, I can still add "comments" and "like" features in the future.
